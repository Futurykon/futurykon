## Agent skills

### Issue tracker

Issues live in GitHub Issues on `Futurykon/futurykon` (uses `gh` CLI). See `docs/agents/issue-tracker.md`.

### Triage labels

Default five-label vocabulary (needs-triage, needs-info, ready-for-agent, ready-for-human, wontfix). See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout — one `CONTEXT.md` and `docs/adr/` at the repo root. See `docs/agents/domain.md`.

### Definition of Done

Shared merge bar plus the per-issue testing and regression policy (`test-infra` and `regression` labels). Every `ready-for-agent` issue carries its own Test scenarios. See `docs/agents/definition-of-done.md`.
